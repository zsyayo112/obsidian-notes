# L0 · 模型是无状态的

> [!abstract] 核心
> **对话历史是你自己维护的一个 Python list。你往里 append 什么，模型就"记得"什么。** 模型本身在两次 HTTP 请求之间什么都不记得。

## 三个实验

1. 最简单的一次调用：`messages` 是一个 list，每项一轮发言，`role ∈ {system, user, assistant}`。这是你和模型之间**唯一**的通信格式，再复杂的 agent 也没变过。
2. 全新调用里问"我刚才问了你什么"→ 模型答不上，因为请求之间它不记得。
3. 手动把历史拼进 messages → 模型立刻"想起来"了。

## 意外收获

实验 2 里模型没说"我不知道"，而是把眼前唯一那句话当答案编了一通——这就是 [[幻觉 模型看不见就编]] 的第一个标本。

## 带走

- ChatGPT 看起来有记忆，是因为它后端每次把完整聊天记录重新发了一遍。没有别的机制。
- 后面整个 agent，本质就是**不断往这个 list 里 append 东西**。
